## 📖 Обзор
Измерение расстояния с помощью ультразвуковых датчиков, таких как **HC-SR04** или **HY-SRF05**, часто дает «шумные» данные и прыгающие значения. **d9dist** решает эту проблему, внедряя встроенную фильтрацию сигнала и упрощенный API, что делает ваши робототехнические проекты намного надежнее.

## ✨ Ключевые особенности
- **🛡️ Фильтрация шумов:** Алгоритм экспоненциального скользящего среднего для устранения случайных скачков данных.
- **🤖 Умная логика:** Метод `near(см)` позволяет мгновенно проверять близость объекта одной строкой.
- **⚡ Без зависаний:** Оптимизированные таймауты (30 мс) гарантируют, что ваш код никогда не заблокируется, если датчик смотрит в пустоту.
- **💎 Чистый код:** Больше никаких ручных вычислений и обработки `pulseIn()` в вашем основном скетче.

## 🛠 Подключение
- **VCC:** 5V
- **Trig (Trigger):** Цифровой пин (например, D12)
- **Echo:** Цифровой пин (например, D11)
- **GND:** Земля (Ground)
- **OUT:** Не используется (оставьте неподключенным)

## 🚀 Быстрый старт
```cpp
#include <d9dist.h>

d9dist sensor(12, 11); // Trig, Echo

void setup() {
  Serial.begin(9600);
}

void loop() {
  // Проверяем, находится ли объект ближе 15 см
  if (sensor.near(15)) {
    Serial.println("Объект слишком близко!");
  } else {
    Serial.print("Расстояние: ");
    Serial.print(sensor.getDist());
    Serial.println(" см");
  }
  
  delay(100);
}
