# d9servo 🚀
**Легкая и интуитивно понятная библиотека Arduino для безупречного управления сервоприводами.**

## 📖 Обзор
Стандартное управление сервоприводами в Arduino часто требует ручного масштабирования и постоянных вызовов `write()`. **d9servo** упрощает это, предоставляя выделенные методы для разных типов сервоприводов, делая ваш код чище и понятнее.

## ✨ Ключевые особенности
- **Упрощенный интерфейс:** Специальные методы для сервоприводов 180° и 360°.
- **Авто-маппинг:** Больше не нужно использовать функцию `map()` в основном скетче для управления скоростью.
- **Безопасность:** Встроенные ограничения для предотвращения выхода за пределы допустимых значений.
- **Энергосбережение:** Удобный метод `detach()` для устранения "гудения" и экономии заряда.

## 🛠 Подключение
- **Сигнал (Оранжевый/Желтый):** Цифровой пин (например, D9)
- **VCC (Красный):** 5V
- **GND (Коричневый/Черный):** GND



## 🚀 Быстрый старт
```cpp
#include <d9servo.h>

d9servo myservo(9); 

void setup() {
  myservo.init();
}

void loop() {
  myservo.rot(90);    // Поворот на 90 градусов
  delay(1000);
  myservo.speed(100); // Полная скорость (для серво 360°)
  delay(1000);
}
